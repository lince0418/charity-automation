## 每月例行檢查清單

| 階段 | 檢查項目 | 完成標準 |
|---|---|---|
| 月初資料確認 | 日期、類別、講師、頭銜、主題 | 五個欄位齊全，沒有待確認文字 |
| 圖片製作 | 月預告、教練講座、成功人士圖片 | 圖片命名符合 `YYYY-MM` 或 `YYYY-MM-DD_類別_講師` |
| 文案製作 | 月預告、預告文、上線日通知 | 已產出到當月 `待發送包` |
| 測試發送 | LINE 官方帳號測試群組 | 先發到「自己喔」並確認圖片與文字一致 |
| 正式發送 | LINE 社群或指定群組 | 登記發送日期、群組、負責人與是否完成 |

## 建議補強欄位

| 欄位 | 用途 |
|---|---|
| 正式發送日期 | 方便回查每月是否真的完成發送 |
| 發送負責人 | 區分孔星元、組員、系統測試或人工轉貼 |
| Canva 最終確認 | 避免圖片已產出但照片排版尚未核定 |
| 影片或上線連結 | 上線日通知前最後檢查用 |
| 異常紀錄 | 紀錄圖片網址失效、群組未收到、文案需重改等問題 |
