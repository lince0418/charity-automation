# 自動化運作地圖

## 主流程

| 階段 | 自動化程度 | 主要檔案 |
|---|---|---|
| 試算表資料讀取 | 自動 | `02_自動化工具/generate_sop_docs.mjs` |
| 四種文案產生 | 自動 | `協會演講/2026年/月份/自動產生文案.md` |
| Canva 完成度檢查 | 半自動 | `00_總覽與日誌/Canva日誌.md` |
| 圖片上傳 Google Drive | 手動 | `03_輸出與測試/LINE圖片連結管理表.md` |
| Drive 圖片網址測試 | 自動 | `02_自動化工具/drive_image_url_test.mjs` |
| LINE 測試群組推播 | 自動 | `02_自動化工具/simulate_month_line_workflow.mjs` |
| 手機提醒 | 匯入後自動 | `03_輸出與測試/Google日曆提醒` |
| 工作日誌 | 自動追加 | `00_總覽與日誌/SOP文檔工作日誌.md` |

## 發送規則

| 文案 | 時間 | 圖片需求 |
|---|---|---|
| 月預告 | 當月月底 19:00 | 月預告圖 |
| 教練講座預告 | 發送日當週星期一 18:00 | 該講座圖 |
| 成功人士預告 | 發送日當週星期一 18:00 | 該成功人士圖 |
| 上線日通知 | 活動當天 19:00 | 該主題圖 |

## 人工確認點

| 項目 | 原因 |
|---|---|
| Canva 人物照片排版 | 需要人工視覺判斷 |
| Google Drive 圖片上傳 | 目前無可用自動上傳工具 |
| 正式 LINE 社群發送 | 私人帳號與 LINE 社群不適合自動代發 |

## 省流量原則

| 情境 | 執行方式 |
|---|---|
| 一般月份模擬 | 使用本機快照，不讀雲端 |
| 需要檢查最新試算表 | 才加上 `--refresh-sheet` |
| LINE 測試 | 先乾跑，確認任務與缺圖狀態後再 `--send-test` |
| 圖片測試 | 只測缺少或剛更新的圖片網址 |
