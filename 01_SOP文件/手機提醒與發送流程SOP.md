# 手機提醒與 LINE 社群發送流程 SOP

## 核心結論

你目前最便利、也最穩的方式是：

> Codex 自動檢查資料與備稿，Google 日曆負責手機提醒，你用手機從待發送包複製文字並上傳圖片到 LINE 社群。

因為你不一定在電腦前，單靠電腦提醒不夠穩。手機提醒建議交給 Google 日曆或手機行事曆。

## 固定發送時間

| 情境 | 發送時間 | 發送內容 |
|---|---|---|
| 月預告 | 當月月底 19:00 | 下個月月預告文字 + 月預告圖片 |
| 講座預告 | 發送日當週星期一 18:00 | 該場預告文字 + 該主題圖片文宣 |
| 上線日通知 | 活動當天 19:00 | 上線日通知文字 + 該主題圖片文宣 |

## 建議提醒方式

| 工具 | 用途 | 優點 | 限制 |
|---|---|---|---|
| Codex 自動化 | 檢查資料、建立待發送包、寫日誌 | 可以讀資料夾與工作日誌 | 不一定會直接推到手機 |
| Google 日曆 | 手機提醒你發送 | 手機最穩、可重複提醒 | 需要建立行事曆事件 |
| LINE 社群 | 實際發送 | 你目前已習慣 | 仍需手動貼文與上傳圖片 |

## 最便利流程

| 時間點 | 系統處理 | 你要做的事 |
|---|---|---|
| 每天上午 | 檢查 Google 試算表與 Canva 狀態 | 看到通知後確認是否要請組員補圖或補資料 |
| 發送前 | 建立待發送文字與圖片清單 | 用手機打開對應資料夾或 Canva |
| 發送時間 | 提醒你發送 | 複製文字、上傳圖片、貼到 LINE 社群 |
| 發送後 | 工作日誌保留待補狀態 | 回到資料夾註記已完成，或請 Codex 幫你更新 |

## Canva 檢查標準

| 檢查項目 | 判斷方式 |
|---|---|
| 日期 | Canva 頁面或 Google 試算表中有活動日期 |
| 主題 | Canva 頁面或 Google 試算表中有主題 |
| 講師 | Canva 頁面或 Google 試算表中有講師 |
| 人物照片排版 | 系統可提醒你確認，但最後建議人工看圖確認 |

## 圖片下載與存放規則

你或組員從 Canva 下載 `.png` 或 `.jpg` 後，建議放到：

```text
協會演講/2026年/月份/宣傳素材/
```

檔名建議：

```text
2026-07-09_教練講座_楊家欣.png
2026-07-23_成功人士_邱凡鳴.jpg
2026-07_月預告.png
```

## 待發送包建議

每次要發 LINE 前，系統整理成：

```text
協會演講/2026年/月份/待發送包/
```

內容包含：

| 檔案 | 用途 |
|---|---|
| `發送文字.md` | 手機直接複製到 LINE 社群 |
| `圖片檢查清單.md` | 告訴你應該附哪一張圖 |
| 圖片檔 | 你下載後放入，方便發送時找到 |

## 手機端建議操作

1. 收到 Google 日曆提醒。
2. 打開手機中的雲端資料夾或 Canva。
3. 複製 `發送文字.md` 內容。
4. 到 LINE 社群確認發送身份或頭銜正確。
5. 上傳對應圖片。
6. 貼上文字並送出。
7. 回來請 Codex 更新工作日誌為已發送。

## 注意事項

- 不建議使用私人 LINE 帳號自動代發。
- 圖片是否完成，建議仍由你做最後確認。
- 如果 Canva 圖片尚未下載到本機，系統可以提醒，但不能保證手機 LINE 能直接抓到本機電腦檔案。
- 最穩做法是把下載圖片放在可被手機同步的資料夾，例如 iCloud Drive、Google Drive 或 Dropbox。
