# LINE 圖片連結管理表

## 使用方式

1. 將本機圖片上傳到可公開讀取的雲端位置。
2. 取得 HTTPS 圖片網址。
3. 填入「公開圖片網址」欄位。
4. 請 Codex 測試 LINE 圖片推播。
