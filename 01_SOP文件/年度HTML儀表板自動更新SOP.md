# 年度 HTML 儀表板自動更新 SOP

## 目的

讓「線上公益講座年度進度儀表板」可以依 Google 試算表分頁「雙周講師SOP」與專案資料夾，自動更新全年 12 個月進度。

## 更新入口

| 更新方式 | 用途 |
|---|---|
| Apps Script 網頁版 | 建議使用；可在 Google 授權環境內即時讀取「雙周講師SOP」 |
| `02_自動化工具/dashboard_web_app.gs` | Apps Script 網頁版原始碼 |
| 雙擊 `更新年度儀表板.command` | 只適合試算表已開放公開 CSV 時使用 |
| 執行 `02_自動化工具/generate_dashboard.mjs` | 技術維護或除錯時使用 |

## 資料來源

| 資料 | 位置 | 用途 |
|---|---|---|
| 雙周講師SOP | Google 試算表 `1GXGEDEoWd0EwYLKrR9Jx6lhubvtzP1SYkT8ZyAyQiCA`，gid `997862433` | 年度講座日期、類別、講師、頭銜、主題 |
| 雙周講師SOP 授權讀取 | `02_自動化工具/dashboard_web_app.gs` | 在 Apps Script 中即時讀取私有 Google 試算表 |
| 圖片網址紀錄 | `03_輸出與測試/LINE圖片連結管理表.md` | 判斷圖片網址是否已建立 |
| 月份資料夾 | `協會演講/2026年/` | 判斷月份素材、文案與待發送包 |
| Apps Script 網頁輸出 | 部署後的網頁應用程式網址 | 即時 HTML 儀表板 |

## 自動判斷規則

| 項目 | 規則 |
|---|---|
| 6 月以前 | 發送完成、上架完成、發送負責人為孔星元 |
| 發送負責人 | 固定顯示孔星元 |
| 圖片進度 | 依圖片連結管理表與月份圖片檔判斷 |
| 文案完成 | 依 `自動產生文案.md` 與 `待發送包` 判斷 |
| 異常紀錄 | 紀錄圖片網址失效、群組未收到、文案需重改等問題 |
| 必要欄位檢查 | `日期`、`類別`、`講師`、`頭銜`、`主題` 必須存在 |

## 注意事項

- 如果 Google 試算表「雙周講師SOP」不是公開 CSV，本機 Node 版會失敗，這是權限問題，不應改用快照。
- 私有試算表建議使用 Apps Script 網頁版，因為它可以在 Google 授權環境中即時讀取資料。
- 本機 `00_總覽與日誌/2026線上公益講座一頁式進度儀表板.html` 已改為停用提示頁，避免快照資料造成誤判。
- 正式查看請使用 Apps Script 部署後的網頁應用程式網址。

## Apps Script 建置方式

1. 打開 Google 試算表「2026公益演講SOP」。
2. 到「擴充功能」→「Apps Script」。
3. 貼上 `02_自動化工具/dashboard_web_app.gs` 的內容。
4. 部署為網頁應用程式。
5. 執行身分選擇自己，存取權依需求設定。
6. 開啟部署網址，即可看到即時年度儀表板。
